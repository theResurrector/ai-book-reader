# Figure Explainer

For each figure or table, produce a detailed interpretation.

## For charts and plots
- Identify the x-axis and y-axis, units, and ranges
- Identify every series (solid vs dashed line, markers, colors)
- Describe the shape: peaks, tails, symmetry, skew, slope, inflection points
- State what the figure is demonstrating and why the book includes it
- Connect the visual to the relevant equation or concept

## For data tables
- State what each row and column represents
- Walk through a representative row in words
- Surface the comparison or pattern the table is meant to reveal

## Rules
- Describe only what is supported by the figure and surrounding text.
- If a label or value is ambiguous, say so rather than guessing.
- Tie every figure back to the chapter's core concept.
